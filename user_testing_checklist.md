## 功能1: 策略回测测试

### 基础回测测试
- [ ] 访问"策略回测"页面
- [ ] 点击"双均线策略"卡片
- [ ] 使用默认参数
- [ ] 点击"🚀 开始回测"按钮
- [ ] 等待结果显示 (约5-10秒)
- [ ] 检查结果表格是否正常显示
- [ ] 验证数据完整性 (收益率、夏普比率、最大回撤、交易次数)

### 参数调整测试
- [ ] 点击"布林带策略"卡片
- [ ] 调整"均线周期"滑块到15
- [ ] 调整"标准差倍数"滑块到1.5
- [ ] 点击"🚀 开始回测"
- [ ] 对比不同参数的结果差异

### 多策略回测测试
- [ ] 同时选择3个策略 (双均线、布林带、动量)
- [ ] 点击"🚀 开始回测"
- [ ] 验证3个策略的结果都正确显示
- [ ] 比较各策略的表现

### 边界条件测试
- [ ] 测试不选择策略直接点击回测 (应显示错误提示)
- [ ] 测试极端参数值 (如快线周期=100)
- [ ] 测试无效日期格式
