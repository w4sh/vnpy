## 已实现策略列表

### 1. 双均线策略 (Dual Moving Average)
- 文件：dual_ma_strategy.py
- 类型：趋势跟踪策略
- 逻辑：快线上穿慢线买入，下穿卖出
- 状态：✅ 已验证，可正常回测
- 适用：趋势明显的市场

### 2. Dual Thrust策略 (双驱动策略)
- 文件：dual_thrust_strategy.py
- 类型：突破策略
- 逻辑：
  - 上轨 = 昨高 + k1 × (昨高 - 昨低)
  - 下轨 = 昨低 - k2 × (昨高 - 昨低)
  - 突破上轨做多，突破下轨做空
- 参数：k1=0.3, k2=0.3, 初始天数=10, 固定仓位=100
- 状态：✅ 已验证，表现优异（23.09%收益）

### 3. 布林带策略 (Bollinger Bands)
- 文件：bollinger_bands_strategy.py
- 类型：均值回归策略
- 逻辑：
  - 中轨 = N日移动平均
  - 上下轨 = 中轨 ± k倍标准差
  - 触及下轨（20%位置）买入，触及上轨（80%位置）卖出
- 参数：周期=20, 倍数=2.0, 仓位比例=20%
- 状态：✅ 已验证，表现优异（19.62%收益，最佳夏普比率0.29）

### 4. 动量策略 (Momentum)
- 文件：momentum_strategy.py
- 类型：动量跟踪策略
- 逻辑：
  - 计算N日收益率
  - 正动量超阈值买入，负动量超阈值卖出
  - 持仓时间限制：60天
- 参数：周期=20, 入场阈值=0.5%, 出场阈值=-0.3%, 仓位比例=20%
- 状态：✅ 已验证，表现不佳（-12.68%收益，不适合震荡市场）

## 问题分析

### ✅ 已解决的关键问题

1. **API调用错误（已修复）**
   - 问题：策略直接操作 `self.targets` 字典
   - 解决：使用 `self.set_target()` 和 `self.get_target()` 方法
   - 结果：所有策略都能正常产生交易信号

2. **仓位管理问题（已修复）**
   - 问题：95%仓位导致过度杠杆和爆仓
   - 解决：降低到20%仓位比例
   - 结果：策略稳定性大幅提高

3. **入场阈值过高（已修复）**
   - 问题：布林带±10%、动量2%阈值过于严格
   - 解决：布林带±20%、动量0.5%
   - 结果：交易频率合理提高

## 技术要点

1. **策略继承**：所有策略继承自AlphaStrategy基类
2. **数据管理**：使用AlphaLab管理历史数据
3. **回测引擎**：BacktestingEngine进行历史回放
4. **参数优化**：网格搜索最优参数组合

### ⚠️ 关键API注意事项

**正确的仓位管理方式**：
```python
# ✅ 正确：使用标准API
current_pos = self.get_target(vt_symbol)
self.set_target(vt_symbol, target_volume)

# ❌ 错误：直接操作字典
current_pos = self.targets[vt_symbol]
self.targets[vt_symbol] = target_volume
```

**仓位管理最佳实践**：
- 仓位比例建议：20-30%（避免过度杠杆）
- 风险控制：设置最大回撤阈值
- 信号确认：使用多时间周期过滤
- 动态调整：根据波动率调整仓位大小
