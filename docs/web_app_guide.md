# vn.py 量化交易Web界面使用指南

## 功能概述

vn.py量化交易Web界面提供了一个直观易用的图形界面，整合了策略回测和智能选股功能。无需编程基础，通过点击鼠标即可完成复杂的量化操作。

## 核心功能

### 1. 策略回测
- 支持4种经典策略（双均线、Dual Thrust、布林带、动量）
- 可视化参数调整
- 一键回测执行
- 详细的回测结果展示

### 2. 智能选股
- 5种选股策略（超卖、超买、突破、收缩等）
- 全市场扫描（5000+股票）
- 可自定义选股条件
- 实时选股结果展示

### 3. 策略对比
- 多策略同时回测
- 自动识别最佳策略
- 风险收益对比分析
- 可视化结果展示

## 快速开始

### 1. 安装依赖


打开浏览器，访问：`http://localhost:5000`

## 功能详解

### 📊 策略回测页面

#### 1. 选择策略
- 点击策略卡片选择要回测的策略
- 可以选择单个策略或多个策略
- 选中的策略会高亮显示

#### 2. 配置参数
- **单个策略**：显示详细参数配置界面
  - 双均线：快线周期、慢线周期
  - 布林带：均线周期、标准差周期、标准差倍数
  - 动量：动量周期、入场/出场阈值
- **多个策略**：使用默认参数简化配置

#### 3. 回测设置
- **开始日期**：回测起始日期
- **结束日期**：回测结束日期
- **初始资金**：起始资金（默认100万）

#### 4. 查看结果
回测完成后会显示：
- 总收益率
- 年化收益率
- 最大回撤
- 夏普比率
- 交易次数

#### 5. 结果解读
- **绿色数字**：正向收益，表现良好
- **红色数字**：负向收益，需要优化
- **夏普比率**：>1为优秀，0-1为一般，<0为不佳

### 🎯 智能选股页面

#### 1. 选股策略说明

**超卖策略** 📉
- **信号**：价格触及布林带下轨
- **含义**：股票被超卖，可能存在反弹机会
- **操作**：考虑买入或观望
- **风险**：可能继续下跌

**超买策略** 📈
- **信号**：价格触及布林带上轨
- **含义**：股票被超买，可能需要警惕
- **操作**：考虑获利了结或设置止损
- **风险**：可能继续上涨

**向上突破** 💥
- **信号**：价格向上突破布林带上轨且开口
- **含义**：强势突破，可能开始上涨趋势
- **操作**：考虑追涨
- **风险**：注意假突破

**布林带收缩** 🎪
- **信号**：布林带宽度极窄
- **含义**：波动率低，可能即将有大行情
- **操作**：加入观察列表，等待突破
- **风险**：方向不确定

#### 2. 参数配置
- **价格范围**：筛选股票的价格区间
- **成交量**：最小成交量要求
- **布林带参数**：均线周期、标准差周期、标准差倍数
- **返回数量**：最多显示多少只股票

#### 3. 结果解读
每只股票显示：
- **股票代码**：如 000001.SZSE
- **当前价格**：最新收盘价
- **布林带位置**：0-100%，越低越超卖
- **得分**：综合评分，分数越高机会越大

### 📊 策略对比页面

#### 1. 选择策略
- 至少选择2个策略进行对比
- 建议选择3-4个策略获得更好的对比效果

#### 2. 对比结果
- **最佳收益**：收益率最高的策略
- **最佳夏普**：风险调整收益最好的策略
- **详细对比表**：各策略的完整指标对比

#### 3. 使用建议
- **趋势市场**：优先选择Dual Thrust和动量策略
- **震荡市场**：优先选择布林带和双均线策略
- **稳健投资**：选择夏普比率高的策略
- **激进投资**：选择收益率高的策略

## 实战案例

### 案例1：寻找超卖反弹机会

**步骤**：
1. 进入"智能选股"页面
2. 选择"超卖策略"
3. 设置参数：
   - 价格范围：10-50元
   - 最小成交量：500万
   - 返回数量：10只
4. 点击"开始选股"
5. 分析结果，选择基本面良好的股票

**交易策略**：
- 分批建仓，不要一次性买入
- 止损：-5%
- 止盈：回到布林带中轨或+10%

### 案例2：对比策略表现

**步骤**：
1. 进入"策略对比"页面
2. 选择4个策略进行对比
3. 设置时间范围：最近1年
4. 点击"开始对比"
5. 查看结果，选择表现最好的策略

**决策依据**：
- 夏普比率 > 1：策略优秀
- 最大回撤 < 20%：风险可控
- 收益率 > 10%：盈利能力好

### 案例3：验证策略参数

**步骤**：
1. 进入"策略回测"页面
2. 选择"布林带策略"
3. 调整参数：
   - 均线周期：10天（更敏感）
   - 标准差倍数：1.5（更宽松）
4. 运行回测，对比默认参数
5. 选择表现更好的参数组合

## 最佳实践

### 1. 日常使用流程

**盘前准备**：
```bash
# 每天盘后运行
./start_web.sh

# 访问 http://localhost:5000
# 1. 查看选股结果
# 2. 验证策略表现
# 3. 制定明日计划
```

**决策流程**：
1. **选股**：使用超卖策略找出潜在机会
2. **验证**：用布林带策略回测验证历史表现
3. **对比**：对比多个策略选择最优方案
4. **执行**：制定详细的交易计划
5. **复盘**：盘后总结，调整参数

### 2. 风险控制

**选股后必须做**：
- 查看公司基本面
- 分析行业趋势
- 检查技术形态
- 设置止损止盈
- 控制仓位比例

**禁止行为**：
- 盲目跟随选股结果
- 一次性满仓买入
- 不设置止损
- 追涨杀跌

### 3. 参数优化

**何时需要调整参数**：
- 策略长期表现不佳
- 市场环境发生重大变化
- 想要适应不同品种
- 需要提高/降低风险

**参数调整原则**：
- 一次只调整一个参数
- 小幅度调整，避免过度优化
- 回测验证新参数
- 实盘小额测试

## 常见问题

### Q1: Web界面启动失败？

**A**: 检查以下几点：
1. 虚拟环境是否激活
2. Flask是否安装：`pip install Flask`
3. 数据目录是否存在：`lab_data/daily/`
4. 端口5000是否被占用

### Q2: 选股结果为空？

**A**: 可能的原因：
1. 当前市场不符合策略条件
2. 参数设置过于严格
3. 需要调整选股策略或参数

### Q3: 回测运行缓慢？

**A**: 优化建议：
1. 减少选择的策略数量
2. 缩短回测时间范围
3. 减少回测股票数量

### Q4: 如何保存回测结果？

**A**: 当前版本结果在浏览器中显示，建议：
1. 截图保存重要结果
2. 复制数据到Excel
3. 或使用命令行工具保存详细报告
