# 运行代码质量检查

运行 vn.py 项目的所有代码质量检查,确保代码符合项目规范。

## 执行的检查

1. **ruff 代码检查**: 运行 `ruff check .` 检查代码风格和潜在问题
2. **mypy 类型检查**: 运行 `mypy vnpy` 进行严格的类型检查
3. **pytest 测试(可选)**: 运行 `pytest -m "not slow" --no-cov` 快速测试(非慢速)

## 使用场景

- 在提交代码前验证代码质量
- 在完成功能开发后检查代码规范
- 在 PR 前确保代码符合项目标准

## 命令

```bash
# 运行 ruff 检查
ruff check .

# 运行 mypy 类型检查
mypy vnpy

# 可选:运行快速测试(跳过慢速测试)
pytest -m "not slow" --no-cov -q

# 可选:运行完整测试套件(包括慢速测试)
pytest --cov=vnpy
```

## 注意事项

- 项目要求所有 ruff 检查都必须通过
- 类型检查使用严格模式,不允许未类型化的定义
- 测试是可选的,但建议在修改核心功能后运行
- 如果检查失败,请修复所有错误后再继续
